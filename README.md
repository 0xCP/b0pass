# 百灵快传（B0Pass）

基于Go的高性能“手机电脑超大文件传输神器”、“局域网共享文件服务器”，只需一个文件双击开启。

- ![扫码（手机扫码或获取链接地址）](https://raw.githubusercontent.com/bitepeng/b0pass/master/docs/images/s2.jpg)
- 扫码（手机扫码）或获取链接地址
